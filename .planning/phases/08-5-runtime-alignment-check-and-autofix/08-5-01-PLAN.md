# Plan 08-5-01 — Autofix engine

## Goal

Land `lib/lint/autofix.js` as a **pure function** module with two exports:
`autofixFrame(astNode) → string` and `autofix(text) → string`. No I/O, no globals,
fully unit-testable without spinning up a hook process.

## Tasks

### Task 1 — RED: Write failing tests
Create `tests/autofix.test.js`. Cases:
- `autofixFrame` repairs a frame with one inner line longer than border.
- `autofixFrame` repairs a frame with right-edge `│` past column `W+1`.
- `autofixFrame` is idempotent on already-clean frames.
- `autofixFrame` preserves Unicode content (▲▼ ✓✗ ├──) inside the frame.
- `autofix(text)` walks AST and only touches frame nodes — non-frame text identical.
- `autofix(text)` leaves text without frames unchanged.
- `autofix(text)` is idempotent (autofix(autofix(x)) === autofix(x)).

Run `npm test` — these must FAIL because `autofix.js` doesn't exist yet.

### Task 2 — GREEN: Implement
Write `lib/lint/autofix.js`:

```js
'use strict';
const { parse } = require('./parser');

function visualWidth(line) {
  // strip ANSI, skip combining marks (̀-ͯ), zero-width joiners
  const stripped = line.replace(/\x1b\[[0-9;]*m/g, '')
                       .replace(/[̀-ͯ​-‏﻿]/g, '');
  return [...stripped].length;
}

function autofixFrame(node) {
  // node = {kind: 'frame', top, inner: [...], bottom, indent}
  const inner = node.inner.map(line => line.replace(/^│/, '').replace(/│\s*$/, '').trimEnd());
  const W = Math.max(...inner.map(visualWidth));
  const top = '┌' + '─'.repeat(W) + '┐';
  const bot = '└' + '─'.repeat(W) + '┘';
  const fixed = inner.map(line => {
    const pad = ' '.repeat(Math.max(0, W - visualWidth(line)));
    return '│' + line + pad + '│';
  });
  return [top, ...fixed, bot].join('\n');
}

function autofix(text) {
  const ast = parse(text);
  // Replace each frame node's text span with its fixed version.
  // ast nodes carry {start, end, kind, ...}; iterate in reverse so offsets stay valid.
  let out = text;
  const frames = ast.filter(n => n.kind === 'frame').sort((a, b) => b.start - a.start);
  for (const f of frames) {
    const fixed = autofixFrame(f);
    out = out.slice(0, f.start) + fixed + out.slice(f.end);
  }
  return out;
}

module.exports = { autofixFrame, autofix, visualWidth };
```

Run `npm test` — all autofix tests must PASS. 245+ existing tests must STILL PASS.

### Task 3 — REFACTOR: Polish + edge cases
- If `lib/lint/parser.js` does NOT currently emit `kind: 'frame'` AST nodes, extend it
  to do so. Frame node = `{kind: 'frame', start, end, top, inner, bottom, indent}`.
  Top/bottom borders detected by leading char `┌`/`└` in the line.
- Add edge case: empty frame (`┌┐` / `└┘`) — autofix is no-op, returns input.
- Add edge case: frame nested inside markdown code-block — DO NOT autofix
  (let user-rendered code blocks be).

Run `npm test` again. All green.

  with isolated test cases verifying frame detection on golden fixtures.

  imperfectly. For v1.0 accept slight under-count; full Unicode width handling is post-8.5.
