# Plan 08-5-03 — L09/L08 + golden fixtures

## Goal

Graduate L09 (currently detection-only from quick-task v0.2.7) to autofix-aware:
violations now carry `expected_after_autofix` so the test harness can verify
both detection AND autofix produce the same fixed output. Harden L08 to skip
combining marks (̀-ͯ) and zero-width joiners (‍) when computing
frame width. Add ≥14 golden fixtures across L09/L08.

## Tasks

- positive: bottom border `└─` length mismatch
- positive: empty padding inside frame (line shorter than W)
- negative: clean frame (already aligned)

End-to-end (2 cases):
