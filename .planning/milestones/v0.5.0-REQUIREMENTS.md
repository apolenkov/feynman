# Requirements Archive: feynman v0.5.0 — Verbosity Economy

**Defined:** 2026-05-11
**Shipped:** 2026-05-11
**Core value:** Every response with structure gets an ASCII diagram without the
developer asking — and uses the cheapest visual that still conveys the structure,
with prose verbosity reduced ≥20% vs v0.4.x.
